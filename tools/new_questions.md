# Q600001
[Type] MCQ
[Topic] 厚膜与薄膜电阻的区别
[Prompt] 在贴片电阻的制造工艺中，厚膜电阻通常采用类似“丝网印刷”的工艺，导电层由导电颗粒和绝缘玻璃交织而成，这导致其存在较高的电流噪声和温度系数（TCR）。而薄膜电阻则通过真空磁控溅射工艺形成致密的金属膜，电子移动顺畅，因此几乎没有电流噪声，且TCR极低。请问，在设计ADC参考电压分压等高精度电路时，为了防止温度变化导致系统精度崩溃，我们应该优先选择哪种电阻？

[Answer]
[ "B" ]
[Explanation] 背景库明确指出，薄膜电阻的温度系数（TCR）可以做到极其优秀（±25ppm/℃甚至±5ppm/℃），且几乎没有电流噪声，非常适合高精度环路反馈或ADC参考电压分压。

[Options]
["大","小","高","低"]
[Answer]
["大","小"]

[Explanation] 封装尺寸越大，走线路径越长，寄生电感越大。为了降低ESL对高速信号的影响，应选择0402、0201等更小尺寸封装的电阻。
